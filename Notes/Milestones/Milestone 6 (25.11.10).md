User stories:
The task management should be improved to match the scale of the project
There should be a pipeline for working with animations
The user can create a room of any type with the core objects and functionality
The user can create and submit levels based on a set of rooms
The user can select levels in "browse" and "story" menus and play the level
Editor tools for creating story rooms and levels
